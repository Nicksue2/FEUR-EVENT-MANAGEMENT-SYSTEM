# FEU Roosevelt Event Management System

A centralized web application designed to manage, track, and streamline event registrations across FEU Roosevelt campuses (Marikina, Cainta, and Rodriguez).

**Live Demo:** [https://feur-event-management-system.vercel.app/](https://feur-event-management-system.vercel.app/)

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend & Database:** Supabase (PostgreSQL, Authentication)
* **Hosting:** Vercel

## ✨ Current Features
* **User Authentication:** Secure Sign Up, Log In, and Session management.
* **Dashboard:** View upcoming events filtered by campus.
* **Registration:** Users can register for free or paid events.
* **Order List:** Users can view their successfully registered events.
* **Admin Panel:** Full CRUD (Create, Read, Update, Delete) capabilities for event management.

## 🚧 Pending Features (Roadmap)
* **QR Code Integration:** Unique QR generation per ticket and an Admin Scanner for attendance checking.
* **Payment Gateway:** Online payment processing for paid events.
